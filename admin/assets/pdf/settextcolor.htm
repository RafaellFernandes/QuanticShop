SetTextColor

SetTextColor(int r [, int g, int b])

Descrição

Define a cor usada pelo texto. Ele pode ser informado como componentes RGB ou tons de cinza. O método pode ser chamado antes que a primeira página seja criada e o valor será mantido para as páginas seguintes.

Parâmetros

r
Se g e b são informados, é componente vermelho; caso contrário, indica o tom de cinza. Valores entre 0 e 255.
g
Componente verde (entre 0 e 255).
b
Componente azul (entre 0 e 255).
