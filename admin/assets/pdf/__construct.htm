__construct

__construct([string orientation [, string unit [, mixed size]]])

Descrição

Esta é o construtor da classe. Ele permite que seja definido o formato da página, a orientação e a unidade de medida usada em todos os métodos (exeto para tamanhos de fonte).

Parâmetros

orientation
Orientação padrão da página. Os valores possíveis são (diferenciando maiúsculas e minúsculas):
  • P ou Retrato
  • L ou Paisagem
O valor padrão é P.
unit
Unidade de medida do usuário. Os valores possíveis são:
  • pt: pontos
  • mm: millímetros
  • cm: centímetros
  • in: polegadas
Um ponto é igual a 1/72 polegadas, isto é o mesmo que 0.35 mm (uma polegada são 2.54 cm). Isto é uma unidade muito comum em tipografia; os tamanhos das fontes são medidos por esta unidade.

O valor padrão é mm.
size
O formato usado pelas páginas. Pode ser um dos seguintes valores (diferenciando maiúsculas e minúsculas):
  • A3
  • A4
  • A5
  • Letter
  • Legal
ou um valor personalizado na forma de um array com dois elementos contendo a largura e a altura (expressas na unidade obtida por unit).

O valor padrão é A4.

Exemplo

Exemplo da criação de uma página de tamanho personalizado com 100x150 mm: 
$pdf = new FPDF('P','mm',array(100,150));
