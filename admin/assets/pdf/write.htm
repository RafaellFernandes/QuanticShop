Write

Write(float h, string txt [, mixed link])

Descrição

Este método imprime um texto a partir da posição atual. Quando a margem direita é atingida (ou o caracter \n é encontrado) uma quebra de linha ocorre e o texto continua a partir da margem esquerda. Quando o método finalizar, a posição atual será imediatamente à esquerda do final do texto.
É possível colocar um link no texto.

Parâmetros

h
Altura da linha.
txt
Texto a ser impresso.
link
URL ou identificador retornado por AddLink().

Exemplo

// Iniciando com fonte normal
$pdf->SetFont('Arial','',14);
$pdf->Write(5,'Visit ');
// Colocando um link azul sublinhado
$pdf->SetTextColor(0,0,255);
$pdf->SetFont('','U');
$pdf->Write(5,'www.fpdf.org','http://www.fpdf.org');
