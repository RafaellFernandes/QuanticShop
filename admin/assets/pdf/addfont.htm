AddFont

AddFont(string family [, string style [, string file]])

Descrição

Importa uma fonte TrueType, OpenType ou Type1 e a deixa disponível. Antes, é preciso gerar um arquivo de definição de fonte com a ferramenta makefont.php.
O arquivo de definição (e o próprio arquivo de fonte quando estiver embutido) precisam estar presentes no diretório de fontes. Se não for encontrado, o erro "Could not include font definition file" (Não foi possível incluir o arquivo de definição de fonte) é gerado.

Parâmetros

family
Família da fonte. O nome pode ser escolhido arbitrariamente. Se for um nome de família padrão ele irá sobrepor a fonte correspondente.
style
Estilo da fonte. Os valores possíveis são (maiúsculas e minúsculas são diferenciadas):
  • empty string: regular
  • B: bold
  • I: italic
  • BI ou IB: bold italic
O valor padrão é regular.
file
O arquivo de definição da fonte.
Por padrão, o nome é composto pelo nome da fonte e o estilo, em minúsculas sem espaços.

Exemplo

$pdf->AddFont('Comic','I');
é equivalente a: 
$pdf->AddFont('Comic','I','comici.php');
