Error

Error(string msg)

Descrição

Este método é chamado automaticamente caso ocorra um erro fatal; ele simplesmente dispara uma exceção com a mensagem informada.
Uma classe herdada pode substituí-lo para personalizar a manipulação de erro, mas o método não deve retornar nunca, caso contrário, o documento resultante provavelmente seria inválido.

Parâmetros

msg
A mensagem de erro.
